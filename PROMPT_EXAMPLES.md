# Prompt Examples

Use these prompts when working with this repository.

## Add a skill

```text
Create a new skill folder named "<skill-name>".
Add SKILL.md with clear trigger conditions and workflow.
Add references only if needed.
Register it in SKILLS.md.
```

## Update a skill

```text
Update "<skill-name>/SKILL.md" to include <new behavior>.
Keep SKILL.md concise and move long detail to references/.
Update SKILLS.md only if name/path/use-when changes.
```

## Review skills catalog

```text
Review SKILLS.md and validate that each listed path exists,
each skill has required frontmatter, and descriptions are precise.
```

## Migrate large SKILL.md

```text
Split "<skill-name>/SKILL.md" into concise workflow + references files.
Keep operational detail in references/.
Update links and verify paths.
```
